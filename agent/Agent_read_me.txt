=======================================
READ ME IF YOU ARE NEW TO THIS TOOL !
=======================================

If you are new to this tool, follow the below instructions:

1. The tool consists of "README.txt" and folders agent, web, manager and manual. You are now in folder "agent".
2. "README.txt" consists of the system requirements, installation instructions and front-end access instructions.
3. The "manager" folder consists of the scripts for running the tool.
4. The "web" folder consists of the files related to the Web GUI.
5. The manual consists of step-by-step instructions for installing the pre-requisite softwares for running the tool.
   It consists of user documentation, developer documentation, installation documentation.
